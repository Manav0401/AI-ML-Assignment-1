# Dataset

This folder is intended to store the dataset required for this project.

## Required Dataset

- **File Name:** `insurance.csv`
- **Dataset:** Medical Cost Personal Insurance Dataset
- **Source:** https://www.kaggle.com/datasets/mirichoi0218/insurance

## Instructions

The dataset is **not included** in this repository.
Please download the dataset from the link above and place the `insurance.csv` file inside this folder before running the project.

The folder structure should be:

```
dataset/
├── insurance.csv
└── README.md
```
